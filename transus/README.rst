=================
Connector Transus
=================

Base module of a Transus connector.


Configuration
=============

The cron job "Transus Receive Message" is by default active after installing the module.

In the company form, under the tab "Transus", fill the following settings:

* Client ID
* Client key
* Set Test Mode
* Transus GLN
